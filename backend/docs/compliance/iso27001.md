# ISO 27001 Controls Checklist — SAQYN RABT

This document registers the self-assessment controls mapping for ISO/IEC 27001 Information Security Management.

## A.9: Access Control
- [x] All users must authenticate via Clerk OAuth.
- [x] Role-Based Access Control (RBAC) scopes API tokens.
- [x] Unused keys are revoked after 90 days of inactivity.

## A.10: Cryptography
- [x] All database operations use secure SSL configurations.
- [x] Cloudflare R2 bucket files utilize server-side encryption (SSE-S3).

## A.12: Operations Security
- [x] Distributed tracing (OpenTelemetry) logs latency metrics.
- [x] Automated Jest tests run on CI push pipelines.
